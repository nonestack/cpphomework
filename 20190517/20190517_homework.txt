2. 封装log4cpp,使用起来比较方便，如下所示。

	　输出的日志信息能同时输出到终端和文件

		int main(void)
		{
			cout << "hello,world"<<endl;
			printf("Message\n");

			LogInfo("Message");	//或者   
			LogError("Message");
			LogWarn("Message");
			LogDebug("Message");
		}
	
	加分项：输出的日志信息中最好能有文件的名字，函数的名字及其所在的行号
				//使用单例模式
